Meet App

Features

  • Users can view a list of events from multiple cities, filter events by a specific city, show/hide event details, and specify the number of events to display.
  • Utilizes Google OAuth2 for user authorization and fetches event data via the Google Calendar API.

  • Supports offline usage and can be easily added to a device's home screen.
